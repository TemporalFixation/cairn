## [0.1.0] — 2026-06-26

### Added
- Asset inventory with full CRUD, condition tracking, building/room assignment
- Repair ticket system with photos, parts tracking, and status workflow
- Dark mode with district crimson & gold theme
- Kiosk mode — Check Out, Check In, Rapid Enrollment with barcode scanner support
- Reports — 9 report types with PDF export via window.print()
- Local user management (Staff/Student, User/SuperUser/Admin roles) with CSV import
- Parts inventory with compatible-part filtering and automatic stock decrement
- Admin config — manufacturers + models hierarchy, ticket options, funding sources
- Secondary asset tags — devices found by any historical tag
- Replace Asset Tag tool (Admin → Replace Asset Tag)
- Customizable app name and color scheme (Admin → App Settings)
- Secure first-run admin password generation
- In-app update checker with GitHub releases integration
- Safe database migrations via Prisma Migrate (no data loss on updates)
